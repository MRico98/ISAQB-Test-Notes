# Annotations and Aspect-Oriented Programming

## Annotations (Decorators)
* Decorate functions, classes, methods or properties with metadata
* Evaluated by annotation processor at either compile time or runtime

### Benefits
Provide and abstraction for manage complexity

## Aspect-Orientend Programming (AOP)
* Increases modularity - Separeted cross-cutting concerns from the bussines logic

* Applies DRY plus Soc plus OCP for cross-cutting concerns

## Summary

* Genreative creations if components
* Built-in programming languages features
* Annotations
* Aspect-oriented programming
